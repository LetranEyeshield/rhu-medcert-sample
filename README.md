RHU Medical Certificate System

A web-based system designed for Rural Health Unit (RHU) to manage patients and issue medical certificates efficiently.

This system helps healthcare staff streamline workflows such as patient registration, certificate issuance.

Features

- Patient Management (Create, Update, Search)
- Medical Certificate Issuance
- Medical Certificate History Tracking
- Printable Certificate Layout
- Authentication (NextAuth)
- Optimized Data Fetching (TanStack Query)
- Structured Architecture (feature-based)


Tech Stack
Frontend
- Next.js (App Router)
- React
- Tailwind CSS

Backend
- GraphQL API
- MongoDB + Mongoose

Other Tools
- TanStack Query
- NextAuth.js
